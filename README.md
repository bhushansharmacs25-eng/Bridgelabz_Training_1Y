# Bridgelabz Training - 1st Year

Welcome to my central repository for the 1st-year Bridgelabz training program. This space documents my daily assignments, logic building exercises, and progression in core computer science fundamentals, implemented entirely in **Java**.

## 🎯 Objective
The primary goal of this repository is to track my progress in Object-Oriented Programming (OOP) and Data Structures & Algorithms. It serves as a living ledger of my technical development as a Computer Science and Engineering student at GLA University.

## 🛠️ Tech Stack
* **Language:** Java (100%)
* **Concepts:** Data Structures & Algorithms, OOP, Logic Building
* **Version Control:** Git & GitHub

## 📚 Learning Modules
The repository is structured around core Java programming concepts:

### 1. Basic Logic Building
* Fundamental I/O operations and control flow in Java.
* Pattern printing, number theory basics, and mathematical logic.
* Array manipulation and `String` handling.

### 2. Object-Oriented Programming (OOP)
* Class design, object instantiation, and method overriding/overloading.
* Practical Java implementations of Inheritance, Polymorphism, Encapsulation, and Abstraction.

### 3. Data Structures & Algorithms
* Foundational algorithmic problem-solving targeting competitive programming standards.
* Implementation of concepts aligned with comprehensive DSA roadmaps (such as Striver's A2Z).
* Linear data structures, sorting, and searching algorithms using both custom classes and the Java Collections Framework.

## 🚀 Future Scope
While this repository currently focuses on establishing a strong foundation in Java and core CS concepts, these logic-building exercises act as a critical stepping stone. The ultimate goal is to leverage these programming fundamentals to build scalable pipelines and eventually integrate them with broader interests in fields like Artificial Intelligence, Machine Learning, and Computer Vision.
